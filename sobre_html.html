Sobre HTML

1 Mas o que é HTML?

Imagem retirada de files.tecnoblog.net

HTML é conhecida como uma linguagem de marcação usada para desenvolver as estruturas de páginas da web visualizadas através dos diversos tipos de navegadores, como Firefox, Brave e Microsoft Edge. A estrutura de uma página da web a partir do HTML é construída e organizada através de tags, essas tags possuem funções e atributos capazes de enviar instruções ao navegador para que o usuário final possa visualizar e interagir com a página em sua tela.

2 A história do HTML

A linguagem de marcação HTML, foi criada pela decada de 1990 pelo físico Tim Berners-Lee e alguns colegas, o HTML o ajudava na formatação de documentos de pesquisa. Com o passar do tempo a internet se expandiu e a lingagem de marcação passou a ter grande utilidade para a criação de páginas na web, principalmente porque as primeiras versões dela eram consideravelmente mais simples, poucas regas de sintaxe tornavam-na ainda mais amigável para a época. A versão 2.0 do HTML foi lançada em 1995, em seguida a 3.2 em 1997. Deste ponto, HTML passou a ter envolvimento do World Wide Web Consortium (W3C), para determinar um padrão do seu uso na internet.
