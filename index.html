A importância de entender as funcionalidades das tags

Desenvolver uma página bem organizada, torna-se mais simples quando se entende bem como as tags funcionam e como se deve utilizá-las. As tags estão presentes em cada linha de código do HTML*, basta um simples "inspecionar elemento" no navegador para que seja possível observar a existência delas. Nos próximos tópicos serão apresentadas algumas das principais tags e seus atributos responsáveis por dá vida à uma página da web.

"HTML: As tags não são apenas enfeites, são a substância do conteúdo." - Desconhecido.

1 Tags: h1, h2, h3, h4, h5 e h6

(voltar ao início)

Essas tags são utilizadas para organização do texto de uma página de forma hierárquica, isto é os conteúdos ficam organizados entre os de maior importância(h1) e os de menor importância(h6), basicamente o que se faz é dividir títulos e subtítulos. Sendo assim descreveremos um pouco mais sobre elas logo abaixo:

Imagem de tags retirada de knoow.net

1.1 h1

No caso da tag h1, normalmente é usada para identificar o título principal de uma página, sendo assim é utilizada somente uma vez e na parte superior da página para delimitar o tema principal de um determinado assunto.

1.2 h2

Usado para separar em tópicos os príncipais assuntos relacionados ao tema príncipal(h1), com esta tag é possível organizar uma página com subtítulos(h2), dividindo o tema principal em vários subtemas a fim de enrriquecer o assunto abordado.

1.3 h3

Geralmente são subtítulos menos importantes que os subtítulos usados no h2, estes organizados de forma hierárquica ficam logo abaixo do h2.

1.4 h4, h5 e h6

Estas últimas tags são utilizadas para subtítulos menos importantes seguindo a mesma ordem de hierárquia. Basicamente é raro o uso delas, depende muito da forma com o texto escrito é organizado.

6 Tags: ol, ul e li

(voltar ao início)

6.1 ol

Tag utilizada para criar um lista ordenada, a lista ordenada ganha um número conforme adicionamos mais elementos a ela. É necessário adicionar uma tag filha para sua funcionalidade correta, essa tag é conhecida como li, que veremos logo mais.

Imagem retirada de i.ytimg.com. Perceba a tag ol sendo utilizada na imagem e seu resultado ao lado.

6.2 ul

Neste caso a lista criada utilizando-se esta tag se faz na forma de um símbolo à frente de cada tópico ao invés de números ordenados de maneira crescente. Para o uso correto também é necessário usar a tag filha li.

Imagem retirada de www.w3resource.com

6.3 li

Esta é a tag filha usada em conjunto com as tag ol e ul para que o texto se organize dentro de uma determinada listagem.

Imagem retirada de www.scaler.com

9 Tags: sub e sup

(voltar ao início)

9.1 sub

Serve para criar uma palavra menor ao lado de um palavra escrita de tamanho normal, é como se a palavra escrita com essa tag se posicionasse na parte de inferior da linha em que o texto se encontra.

Imagem retirada de miro.medium.com

9.2 sup

De forma contrária ao sub, na tag sup a palavra continua menor em relação ao texto mas desta vez se posicionando na parte superior da linha em que o texto se encontra.

Imagem retirada de miro.medium.com

11 Tags extras

(voltar ao início)

As tags apresentadas a seguir, também tem bastante utilidade na hora de desenvolver uma página em HTML. São elas:

11.1 font

Esta tag é usada para alterar três atributos de textos, como o tipo de fonte, tamanho dela e cor do texto. Veremos estes atributos a seguir:

Imagem retirada de i.ytimg.com

Observe que todos os atributos mencionados abaixo, estão exemplificados na imagem acima.

11.1.1 face

Este atributo é colocado junto da tag font para possiblitar a troca do tipo de fonte, como Arial e Times New Roman por exemplo.

11.1.2 size

Este atributo é responsável pela alteração do tamanho da fonte de uma palavra ou texto.

11.1.3 color

Por fim este é o atributo que garante a troca de cores de uma palavra ou texto.

11.2 del

Essa tag é utilizada para sinalizar que palavras ou frases foram deletadas de um texto, visualmente falando ela se apresenta como uma linha que risca palavras ao meio nos textos.

11.3 abbr

Tag utilizada para fazer abreviações de palavras, é usada em conjunto com o atributo tittle que ajuda a descrever de forma completa o que uma determinada abreviação significa.

12 Principais Benefícios do HTML

(Voltar ao início)
  1. Alta compatibilidade: HTML é suportado por todos os navegadores modernos, garantindo a exibição de uma página na web em diversos lugares possíveis, atingindo um público considerável.
  2. Estrutura de conteúdo (semântica): faz uso de tags que organizam o conteúdo de uma página na web, permitindo o desenvolvimento correto de estruturo e significado para os diversos elementos presentes na tela.
  3. Acessibilidade: com o uso correto das tags e seus atributos, é possível desenvolver uma estrutura da página na web extremamente inclusiva. Visto que muitas pessoas precisam fazer uso de textos audíveis para conseguirem consumir conteúdos online.
  4. Aprendizado mais simples: o HTML é uma linguagem de marcação simples e intuitiva, o uso de tags e atributos para o desenvolimento de uma página web facilita muito o aprendizado inicial desta linguagem.
